Interactivity in Game Context

Explore the fundamental concepts of game design, playful user experience, and interactivity using Unity.

Course Objectives

This subject introduces the fundamental concepts of game design, playful user experience and interactivity. It aims to enable students to construct narrative interaction designs in the game context based on exploring the principles of interactive storytelling, dynamic game balancing and user psychology.

Intended Learning Outcomes:

  • Describe the fundamentals of interaction design in game context.
  • Apply design skills through prototyping non-digital games.
  • Employ psychological knowledge of human behavior in playing experiences.
  • Outline the history of games to reflect the momentum from relatedness needs.
